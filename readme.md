Readme
=========
AMD for [swing](https://github.com/gajus/swing) - A swipeable cards interface. The swipe-left/swipe-right for yes/no input. As seen in apps like Jelly and Tinder.

Install
==========
```shell
bower install swing-amd --save-dev
```
